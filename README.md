# study_app_firebase

A completely Responsive Study App - works on Android, Web & Desktop! Works as Study App on Smaller Screen Sizes and WhatsApp Web on Bigger Screen Sizes!

# Features

Google Authentication <br>
Two Themes - Light mode and Dark mode. <br>
Quiz from firebase <br>
Shimmer <br>
15 minutes timer for every Quiz. <br>
Review - Correct answer or Wrong.
Saving user and their scores to cloud firestore.


# Tech Used

Server: Firebase Auth, Firebase Storage, Firebase Firestore

Client: Flutter, GetX
